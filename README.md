# ECG Anomaly Detection

This project is part of the course "Geometric Learning, Time-Variant Data Analysis, and Anomaly Detection" and focuses on ECG signal analysis for anomaly detection.

## Project Structure

- `data/raw/` — Original raw data
- `data/processed/` — Preprocessed data ready for analysis
- `notebooks/` — Jupyter notebooks for exploration, prototyping, and reporting
- `utils.py` — Utility functions for preprocessing, visualization, etc.
- `models.py` — Model definitions and training (ARMA, Autoencoder, VAE, LSTM, Liquid Neural Networks, ...)

## Goals
- Explore classical and deep learning techniques for ECG anomaly detection
- Compare performance and interpretability of different models
- Experiment with Liquid Neural Networks for anomaly detection

## Planned Models
- ARMA
- Autoencoder
- Variational Autoencoder (VAE)
- LSTM Autoencoder
- Liquid Neural Networks (LNN)
